# AI DevOps Team Project Template

Welcome! This template automates GitHub project setup with AI agents for backlog management, pipeline orchestration, and DevOps integration. Use this as a starting point for new projects.

## Quickstart

1. **Clone this repo** to your new project directory.
2. **Review and update** `docs/backlog.md` for your features and priorities.
3. **Customize** `docs/architecture.md` to reflect your project’s architecture.
4. **Run the onboarding script** to automate GitHub repo setup (see below).
5. **Start the AI agent workflow** as described in this documentation.

## Features
- Automated backlog and pipeline management
- Modular AI agent workflow (BacklogManager, Conductor, DevOps)
- Local and multi-environment deployment support
- Extensible for new features and infrastructure

## Onboarding Automation

Use the provided script in `scripts/github-setup.sh` to automate initial GitHub repository configuration:
- Create labels
- Set branch protection rules
- Add issue templates

## Customization
- Update `docs/backlog.md` and `docs/architecture.md` for your project
- Add or modify scripts in `scripts/` as needed
- Integrate additional AI agents or workflows as required

## Support
See `docs/dev-notes.md` for development tips and troubleshooting.
